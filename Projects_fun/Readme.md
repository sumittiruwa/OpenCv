# OpenCV Learning

This repository documents my learning journey with OpenCV using Python. It contains practice programs, examples, and mini projects covering the fundamentals of computer vision and image processing.

## Overview

The purpose of this repository is to build a strong foundation in OpenCV by exploring image processing techniques, video analysis, and real-time computer vision applications.

## Topics Covered

* Image Reading and Display
* Image Saving
* Webcam Capture
* Video Processing
* Drawing Shapes and Text
* Image Transformations
* Color Spaces
* Image Thresholding
* Contour Detection
* Edge Detection
* Face Detection
* Object Detection
* Image Filtering
* Feature Detection
* Real-Time Computer Vision Projects

## Technologies Used

* Python
* OpenCV (cv2)
* NumPy

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/sumittiruwa/OpenCv.git
```

### Install Dependencies

```bash
pip install opencv-python numpy
```

### Run a Python File

```bash
python filename.py
```
